Как это работает и демонстрирует паттерн Composite:
FileSystemNode — абстрактный базовый класс (Компонент)

Определяет единый интерфейс для всех элементов

File — Лист (Leaf)

Не может содержать детей

getSize() возвращает размер конкретного файла

Directory — Композит (Composite)

Может содержать детей (другие Directory или File)

getSize() рекурсивно суммирует размеры всех детей

print() рекурсивно выводит всю структуру
